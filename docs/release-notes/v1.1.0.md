# Release Notes - Versão 1.1.0

Facilita o cadastro de estudantes para permitir vínculo a múltiplas turmas.

## Visão Geral

Esta versão permite que estudantes sejam vinculados a múltiplas turmas sem necessidade de recriar cadastro. Quando um estudante recebe um novo convite, o sistema identifica se o usuário já existe e apenas vincula à nova turma, simplificando o processo de matrícula.

---

## Funcionalidades Principais

### Vínculo de Estudantes a Múltiplas Turmas

- Estudantes podem ser vinculados a várias turmas sem recriar cadastro
- Campo `requiresSignup` em `/auth/validate-invite` indica se usuário precisa ser criado ou apenas vinculado
- Quando `requiresSignup=false`, campos `name` e `password` são opcionais em `/auth/complete-signup`
- Endpoint `/auth/me` retorna lista de todas as turmas ativas do estudante (`id` e `className`)

### Melhorias Adicionais

- Validação de usuário ativo no login
- Removido campo `name` do `UserProfileDTO`
- Refatoração de DTOs para usar `@Data`

---

## Informações Técnicas

### Endpoints Alterados

**GET /auth/me**

- Removido: `name`
- Adicionado: `classes` (List<ClassroomInfoDTO>) para estudantes

**GET /auth/validate-invite**

- Adicionado: `requiresSignup` (boolean)

**POST /auth/complete-signup**

- Campos `name` e `password` opcionais

## Notas

- Sem breaking changes
- Sem migrações de banco de dados necessárias
